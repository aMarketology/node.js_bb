Wallet Creation Flow:
1. Generate 12-word BIP-39 mnemonic (128-bit entropy)
2. Derive seed using SLIP-0010 at path m/44'/1337'/0'/0'/0'
3. Create Ed25519 keypair from 32-byte seed
4. Generate address: L1 + SHA256(pubkey).slice(0,40).toUpperCase()

Address Format:
  - Old: L1_ + 14 hex chars (56-bit) - DEPRECATED, still accepted
  - New: L1 + 40 hex chars (160-bit) - Same security as Bitcoin

Domain Separation:
  - CHAIN_ID_L1 = 0x01 (Layer 1 - Bank/Vault)
  - CHAIN_ID_L2 = 0x02 (Layer 2 - Gaming/Betting)
  - L1 signatures CANNOT be replayed on L2 (and vice versa)
